The T17 computer in the common room works as a diary, chat, planner, and generally useful/amuseful thing.

It is an on-going experiment with running a shared home computer based on cheap hardware and free software.

The actual budget was around 25 EUR for the Raspberry Pi B+ computer plus a used 17" Dell monitor for 10 EUR and a used keyboard for 3 EUR.

The software on it is free software, which means that we are free to study, modify, and extend the code.

The software is the GNU system with a Linux kernel and Emacs user interface.

Despite running in console mode, the T17 computer is somewhat user friendly.

Features of the T17 computer

  1. The log (or "diary") lets us write public messages. It uses a custom Emacs mode.

  2. The organizer has stuff like TODOs, checklists, and agenda items. It's based on "Org mode".

  3. The underlying GNU Emacs system is completely customizable through Lisp programming.

  4. There is a clock.
