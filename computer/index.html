The T17 computer in the common room works as a diary, chat, planner, and generally useful/amusing thing.

It is an on-going experiment with running a shared home computer based on cheap hardware and free software.

The base software is Linux, GNU, and GNU Emacs.

Since the whole system is free software, we are able to study and modify the code as much as we want without any license fees or restrictions.

Budget

Raspberry Pi B+ computer 30 EUR
Used 17" Dell monitor with DVI output 10 EUR
Used DVI to HDMI adapter 8 EUR
Used keyboard 3 EUR

Functions

  1. The log (or "diary") lets us write public timestamped messages. It uses a custom Emacs mode.

  2. The organizer has stuff like todos, checklists, and agenda items. It's based on "Org mode".

  3. The underlying GNU Emacs system is completely customizable through Lisp programming.

  4. There is a clock.
